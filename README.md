 Navigation of IntelliJ IDEA
 1. Navigation bar
 2. Search class: Cmd + O / Ctrl + N
 3. Search Symbols: Cmd + alt + O / Ctrl + alt + shift + N
 4. Navigate to reference: Cmd / Ctrl + click
 5. Go to super class: Ctrl + U
 6. Go to line: Ctrl G
 7. Navigate History: Ctrl + Alt + left/right
